# Changelog

## 0.1.22

Released: 2026-06-17

### 新功能

- **Bridge 下载断点续传**：Bridge 更新下载支持断点续传，网络中断后重新下载可从中断处继续，无需重新下载完整文件
- **Bridge 更新使用系统代理**：Bridge 下载更新文件时自动使用系统代理设置，解决代理环境下无法下载的问题

### 问题修复

- **Bridge 更新后状态卡住**：修复 Bridge 应用更新后 Android 端一直显示"正在应用更新"的问题，现可自动检测 Bridge 重启完成并恢复状态
- **Bridge 更新脚本超时**：更新脚本等待进程退出时间从 10 秒缩短为 3 秒，超时后强制终止并继续更新，避免更新流程耗时过长
- **Bridge 更新失败回滚**：更新脚本在强制终止旧进程后清理备份文件；Linux 脚本新增 kill -9 兜底机制
- **version.json 源切换**：更新检测源从 jsDelivr CDN 切换为自建 Gateway 服务器（主源），解决 jsDelivr 中国节点缓存导致无法检测新版本的问题
- **Bridge 更新错误提示国际化**：Bridge 更新失败提示改为中文显示，不再显示原始英文错误信息

## 0.1.21

Released: 2026-06-16

### 问题修复

- **设置页中文显示**：Bridge 更新标签改为中文显示
- **更新提示优化**：自动检查更新不再弹出"已是最新"提示，仅用户手动点击检查时才显示
- **开源许可证**：设置页开源许可证条目改为可点击，跳转至 Apache 2.0 许可证页面
- **版本信息去重**：移除关于区域中重复的版本号显示
- **消息渲染**：修复用户消息换行丢失和 Markdown 不渲染的问题

## 0.1.20

Released: 2026-06-16

### 新功能

- **App 自动更新**：设置页新增 App 更新卡片，支持检查新版本、下载并安装 APK（jsDelivr CDN + raw.githubusercontent.com 双源 version.json 发现，tag-based 下载）
- **Bridge 自动更新**：Bridge 支持自更新（下载、校验、helper-script 替换、失败回滚）；Android 设置页可触发 Bridge 更新并查看进度

## 0.1.19

Released: 2026-06-16

### 新功能

- **Linux 兼容性大幅提升**：Bridge Linux 版本改为静态链接（musl），不再依赖系统 glibc 版本，可在任意 Linux 发行版上运行（Ubuntu/Debian/CentOS/Alpine 等均可）
- **新增 Linux arm64 支持**：Bridge 提供 arm64 架构预编译二进制，支持 ARM 服务器与设备

### 问题修复

- 修复网络断开后重连时可能崩溃的问题（No active database）

## 0.1.17

Released: 2026-06-12

### 新功能

- **EventReplayer V2 重写**：适配 opencode 1.17 V2 事件流，重写增量同步核心逻辑，提升同步稳定性
- **OPENCODE_EXPERIMENTAL 自动设置**：Bridge 启动时自动为当前用户持久化设置环境变量

### 问题修复

- **Compaction 消息**：修复 compaction 开始时不显示、完成后一直计时的问题
- **上下文使用量**：修复 opencode 升级后 tokens.total 字段缺失导致上下文使用量不显示
- **Task 子代理导航**：修复 V2 事件流中 task 子代理点击后无法跳转到子会话的问题
- **Unrevert 恢复**：修复 opencode unrevert 后消息不恢复显示的问题
- **连接恢复**：修复连接进入 Failed 状态后不再自动重试的问题，加入指数退避自动重连
- **初始化快照**：简化 SQL 查询避免子查询兼容性问题

## 0.1.5

Released: 2026-06-05

### 新功能

- **网关开关**：每个实例可单独开启/关闭网关中继，直连环境下可关闭以减少不必要的探测
- **后台断连**：App 进入后台时主动断开 SSE，前台恢复后立即重连，节省电量
- **快速重连**：缓存上次连接路径，重连时跳过探测直接连接，速度更快
- **列表样式**：实例/工作区/会话列表改为扁平化样式，去掉卡片边框，更简洁

### 问题修复

- 修复连接状态机多处稳定性问题：SSE 重连逻辑、探针时序、状态切换崩溃
- 修复切换实例时连接状态未重置，导致旧连接数据残留
- 修复网关连接时 baseUrl 未动态更新，导致 API 请求走错路径
- 修复 Git 工具处理非 ASCII 文件名时乱码（octal 编码解码）
- 修复 Git 命令无限递归导致 Bridge 崩溃（栈溢出）
- 修复 Windows 上 Git 进程弹出命令行窗口
- 修复权限请求同时弹出多个卡片的问题

## 0.1.4

Released: 2026-05-28

### 新功能

- **Git 变更查看器**：新增 Bridge Git API，支持查看文件差异（DiffViewer）和独立文件浏览器
- **QR 登录**：支持浏览器扫码登录认证
- **Diff 查看器**：点击编辑工具标题可直接查看 diff，新增查看文件菜单

- **Linux 无头配对**：Bridge 终端 QR 码配对，支持 openmate:// 自定义协议
- **Linux 部署**：Bridge systemd 服务模式，TCP keepalive
- **自启动**：Bridge 支持 autostart_mode，Linux 自动启动

### 问题修复

- 修复 Android 长文本/推理内容导致 Compose 布局崩溃（高度溢出）

## 0.1.3

Released: 2026-05-24

### 新功能

- **远程配对**：支持扫码配对，手机扫描 Bridge 二维码即可连接，无需手动输入地址
- **网关中继**：新增 Relay Gateway，局域网不可用时自动通过云端网关中继连接
- **连接状态机**：全面重构连接管理，网络切换、断连重连更稳定可靠
- **会话初始化**：新增"初始化"菜单，一键生成 AGENTS.md 项目配置
- **同步日志查看器**：新增同步调试日志面板，支持按类别过滤、复制和清除
- **崩溃日志**：应用崩溃时自动采集日志，支持查看、复制和分享
- **Bridge 管理 UI**：Bridge 新增 Web 管理界面，可查看设备、日志和网络状态
- **Bridge 配置 API**：支持运行时修改端口等配置，端口冲突时自动回退
- **Bridge 系统托盘**：Bridge Windows 版支持最小化到系统托盘，后台静默运行

### 问题修复

- 发送消息时，仅在 API 成功后才清空输入框，失败时保留内容，发送过程显示进度条
- 修复 SSE 心跳保活被网关 keepalive 注释行干扰导致连接断开
- 修复旧 SSE 状态覆盖新连接，统一连接状态来源
- 修复网关转发时 query string 和 gzip 编码丢失问题
- 修复 Bridge 连接网关时 TLS 握手静默失败
- 修复 Bash 工具长输出截断丢失描述字段的问题
- 修复 QR 扫描时 token 过期导致配对失败
- 所有错误提示改为国际化显示，不再暴露技术异常信息
