# Changelog

## 0.1.5

Released: 2026-06-05

### 新功能

- **网关开关**：每个实例可单独开启/关闭网关中继，直连环境下可关闭以减少不必要的探测
- **后台断连**：App 进入后台时主动断开 SSE，前台恢复后立即重连，节省电量
- **快速重连**：缓存上次连接路径，重连时跳过探测直接连接，速度更快
- **列表样式**：实例/工作区/会话列表改为扁平化样式，去掉卡片边框，更简洁

### 问题修复

- 修复连接状态机多处稳定性问题：SSE 重连逻辑、探针时序、状态切换崩溃
- 修复切换实例时连接状态未重置，导致旧连接数据残留
- 修复网关连接时 baseUrl 未动态更新，导致 API 请求走错路径
- 修复 Git 工具处理非 ASCII 文件名时乱码（octal 编码解码）
- 修复 Git 命令无限递归导致 Bridge 崩溃（栈溢出）
- 修复 Windows 上 Git 进程弹出命令行窗口
- 修复权限请求同时弹出多个卡片的问题

## 0.1.4

Released: 2026-05-28

### 新功能

- **Git 变更查看器**：新增 Bridge Git API，支持查看文件差异（DiffViewer）和独立文件浏览器
- **QR 登录**：支持浏览器扫码登录认证
- **Diff 查看器**：点击编辑工具标题可直接查看 diff，新增查看文件菜单
- **优雅关机**：Bridge 支持 Ctrl+C 优雅停止
- **Linux 无头配对**：Bridge 终端 QR 码配对，支持 openmate:// 自定义协议
- **Linux 部署**：Bridge systemd 服务模式，TCP keepalive
- **自启动**：Bridge 支持 autostart_mode，Linux 自动启动

### 问题修复

- 修复 Android 长文本/推理内容导致 Compose 布局崩溃（高度溢出）

## 0.1.3

Released: 2026-05-24

### 新功能

- **远程配对**：支持扫码配对，手机扫描 Bridge 二维码即可连接，无需手动输入地址
- **网关中继**：新增 Relay Gateway，局域网不可用时自动通过云端网关中继连接
- **连接状态机**：全面重构连接管理，网络切换、断连重连更稳定可靠
- **会话初始化**：新增"初始化"菜单，一键生成 AGENTS.md 项目配置
- **同步日志查看器**：新增同步调试日志面板，支持按类别过滤、复制和清除
- **崩溃日志**：应用崩溃时自动采集日志，支持查看、复制和分享
- **Bridge 管理 UI**：Bridge 新增 Web 管理界面，可查看设备、日志和网络状态
- **Bridge 配置 API**：支持运行时修改端口等配置，端口冲突时自动回退
- **Bridge 系统托盘**：Bridge Windows 版支持最小化到系统托盘，后台静默运行

### 问题修复

- 发送消息时，仅在 API 成功后才清空输入框，失败时保留内容，发送过程显示进度条
- 修复 SSE 心跳保活被网关 keepalive 注释行干扰导致连接断开
- 修复旧 SSE 状态覆盖新连接，统一连接状态来源
- 修复网关转发时 query string 和 gzip 编码丢失问题
- 修复 Bridge 连接网关时 TLS 握手静默失败
- 修复 Bash 工具长输出截断丢失描述字段的问题
- 修复 QR 扫描时 token 过期导致配对失败
- 所有错误提示改为国际化显示，不再暴露技术异常信息
