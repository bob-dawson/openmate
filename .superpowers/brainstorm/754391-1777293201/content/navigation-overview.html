OpenMate 导航结构

底部 Tab 导航 + 栈式深入

底部 Tab 栏（3个主 Tab）

Tab 导航概览
💻
实例
💬
会话
⚙️
设置

Tab 1: 实例

  • 实例列表（在线/离线状态）
  • → 点击实例 → 工作区列表
  • → → 点击工作区 → 该工作区的会话列表
  • → → → 点击会话 → 会话详情（消息流）
  • → 远程启停 opencode

Tab 2: 会话

  • 所有活跃会话的聚合视图（跨实例）
  • 类似聊天应用的会话列表
  • → 点击会话 → 直接进入会话详情
  • 快速切换最近使用的会话

Tab 3: 设置

  • 账户管理（GitHub 登录/登出）
  • 通知设置
  • 外观（主题、字体大小）
  • 关于

页面导航栈
📱 启动
│
├─ 🔐 登录页（首次）
│ └─ GitHub OAuth
│
└─ [Tab 栏]
│
├─ 💻 [实例 Tab]
│ ├─ 实例列表
│ │ ├─ 🟢 在线 / 🔴 离线 标识
│ │ ├─ ▶️ 远程启动 opencode
│ │ └─ ⏹️ 停止 opencode
│ │
│ └─ → 工作区列表
│ ├─ 工作区卡片（名称、分支、状态）
│ ├─ ➕ 创建工作区
│ ├─ 🗑️ 删除工作区
│ │
│ └─ → 会话列表（该工作区内）
│ ├─ 会话卡片（标题、摘要、时间）
│ ├─ ➕ 新建会话
│ ├─ 🗑️ 删除会话
│ │
│ └─ → 会话详情
│ ├─ 💬 消息流
│ ├─ ✏️ 输入框 / 发送
│ ├─ 🔧 工具调用展示
│ ├─ ✅ 权限确认对话框
│ ├─ ❓ 问题回答对话框

│ └─ ⏹️ 中止操作
│
├─ 💬 [会话 Tab]
│ └─ 所有活跃会话（跨实例聚合）
│ └─ → 会话详情（同上）
│
└─ ⚙️ [设置 Tab]
├─ 账户管理
├─ 通知设置
├─ 外观
└─ 关于
